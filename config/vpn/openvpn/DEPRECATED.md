# ⚠️ OpenVPN - DEPRECATED

OpenVPN конфигурация помечена как **устаревшая (legacy)**.

## Рекомендуемая альтернатива

Используйте **Hysteria2** — см. документацию в `../hysteria2/README.md`

## URI для подключения

```
hysteria2://SenpaiCat2407@vpn.vibe-labs.ru:8443/?sni=vpn.vibe-labs.ru
```

## Причины миграции

- Hysteria2 использует QUIC/UDP — лучшая производительность
- Работает как прокси (SOCKS5/HTTP) — проще в настройке
- Не требует привилегий NET_ADMIN для базового использования
- Лучше обходит блокировки

## Поддержка OpenVPN

OpenVPN остаётся доступным как **резервный вариант**, но:
- Новые фичи не добавляются
- Документация не обновляется
- При проблемах — рекомендуется перейти на Hysteria2
